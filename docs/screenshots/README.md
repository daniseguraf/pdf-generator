# Screenshots

Este directorio contiene las capturas de pantalla del proyecto My Buildings.

## Capturas Requeridas

Para completar la documentación del proyecto, se necesitan las siguientes capturas de pantalla:

### 1. Login Page (`login.png`) ✅

- Página de inicio de sesión ✅
- Mostrar el formulario de login con campos de email y password ✅
- Capturar con el diseño limpio y moderno de Mantine UI ✅

### 2. Buildings List (`buildings-list.png`)

- Vista principal con la lista de edificios
- Mostrar al menos 2-3 edificios en la tabla
- Incluir el header con el botón "Nuevo Edificio"
- Mostrar las columnas: Nombre, Dirección, Tipo, Pisos, Estado

### 3. Building Detail (`building-detail.png`) ✅

- Vista de detalle de un edificio específico ✅
- Mostrar las tabs: "Información General", "Áreas Comunes", "Amenities" ✅
- Capturar con la tab de "Áreas Comunes" activa mostrando al menos 2 áreas ✅
- Incluir los botones de editar y eliminar ✅

### 4. Reservations (`reservations.png`)

- Página de reservaciones
- Mostrar el formulario de nueva reservación
- Incluir la lista de "Mis Reservaciones" con al menos 1-2 reservaciones
- Mostrar los estados de las reservaciones (Confirmada, En Revisión)

## Recomendaciones para las Capturas

- **Resolución:** 1920x1080 o similar (pantalla completa)
- **Formato:** PNG para mejor calidad
- **Contenido:** Usar datos de ejemplo realistas (no "test", "asdf", etc.)
- **Navegador:** Chrome o Firefox con DevTools cerrado
- **Zoom:** 100% (sin zoom aplicado)
- **Tema:** Asegurarse de que el tema de Mantine se vea bien

## Herramientas Recomendadas

- **macOS:** Cmd + Shift + 4 (selección de área)
- **Windows:** Snipping Tool o Win + Shift + S
- **Linux:** Flameshot o gnome-screenshot
- **Extensión:** Awesome Screenshot (Chrome/Firefox)
